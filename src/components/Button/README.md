# Button

# 示例

### 基本用法

<!--demo-->

```jsx
import { Button } from 'stbui';

class Demo extends React.Component {
  render() {
    return (
      <div>
        <Button>button</Button>
        <Button raised>button</Button>
        <Button stroked>button</Button>
        <Button fab>button</Button>
      </div>
    );
  }
}

ReactDOM.render(<Demo />, mountNode);
```
